# ❓ Simple FAQ Page – Design Challenge 04

Reproduction d’une page FAQ (Questions Fréquemment Posées) à partir d’un prototype de design.

Ce projet fait partie de ma série de défis d’intégration frontend visant à améliorer mes compétences en création d’interfaces modernes et responsives.

---

## 🚀 Démo en ligne

👉 GitHub Pages : https://charlie38-devlab.github.io/ui-simple-faq-design-challenge-04/

👉 Vercel : https://ui-simple-faq-design-challenge-04.vercel.app/

---

## 📌 Objectif du projet

- Reproduire fidèlement un design UI
- Structurer une page FAQ claire et lisible
- Améliorer mes compétences en HTML et CSS
- Implémenter un responsive design propre

---

## 🛠 Technologies utilisées

- HTML5
- CSS3
- Flexbox
- Responsive Design

---

## 📱 Responsive Design

Le projet est optimisé pour :

- 💻 Desktop
- 📱 Tablette
- 📲 Mobile

---

## 🎯 Compétences travaillées

- Organisation du code HTML
- Gestion des espacements et typographies
- Création de composants UI
- Adaptation responsive multi-écrans

---

## 🔮 Améliorations futures

- Ajouter un accordéon interactif en JavaScript
- Ajouter des animations d'ouverture
- Connecter la FAQ à un backend
